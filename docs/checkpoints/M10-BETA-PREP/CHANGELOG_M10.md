## [M10 Beta Prep] — 2026-06-16

### Added

#### Reserve Duty Module (מילואים)

- New route `/reserve` accessible from Dashboard secondary quick-action row
- New store `src/lib/reserve-store.ts` with separate localStorage key `reserve_os_state` (version 1)
- Service period tracking: start date, end date, inclusive day count
- Document checklist with 4 types:
  - צו גיוס
  - אישור שירות מילואים
  - אישור תקופת שירות
  - אישור תביעה לביטוח לאומי
- Compensation tracking: estimated amount, expected payment date, status (pending / submitted / approved / paid)
- Free-text notes with inline edit
- Compensation summary card visible at top of list (shows total estimated amount + dominant status)
- Empty state with explanatory copy and "צור תקופת שירות" CTA
- Two-step delete with confirmation
- Collapsible period cards showing missing document count badge
- Dashboard: Shield icon card linking to `/reserve` in secondary quick-nav
